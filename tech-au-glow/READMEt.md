# tech-ai-glow

React (Vite) + Tailwind + Framer Motion project — neon glass UI with auto-changing background themes.

## Run locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Build
`npm run build`

## Deploy
Push to GitHub and import project into Vercel (auto-detects Vite).
